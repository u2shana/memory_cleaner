# Memory Cleaner 重制版 Fork
原版 Memory Cleaner 还是非常好用的，但是会强制联网更新，经常连不上他的服务器就用不了了。<br>
此版完全按照原版的内部逻辑重写所有功能，我自己还额外加了清理内存等待链（适用于游戏玩家）。<br>

相比重制版，此版能够：
1、自定义内存占用率阈值。
2、将原本注册表实现的开机启动调整为更稳定的计划任务方式实现。若程序位置调整，需要重新设置。
